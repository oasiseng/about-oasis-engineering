# Wind Package Direct Links

Verified against the live `windcalculations.com` package catalog on March 18, 2026.

Use this page when a wind-related request clearly matches one of the pre-packaged solutions on [https://windcalculations.com/wind-calc-packages](https://windcalculations.com/wind-calc-packages).

## Core Rule

If the request is really a wind engineering scope, send it to the wind lane.

If it also matches one of the package types below, send it to the direct package URL instead of the generic wind contact page.

## Routing Notes

- If the request mentions windows, doors, roofing, or solar and it clearly matches one of the direct package pages above, send the package URL first.
- If the request mentions wind but the scope is broader, custom, multi-building, or unclear, use [https://windcalculations.com/contact](https://windcalculations.com/contact).
- If the request sounds like a letter request but the substance is wind engineering, it is still a wind project.
